# LAB2

## 运行方式

```bash
python3 -m venv .venv
.venv/bin/python -m pip install --upgrade pip
.venv/bin/pip install opencv-python sounddevice soundfile pydub matplotlib
.venv/bin/python main.py
```

## 目录说明

- `assets/`：实验素材，包含 4 张猫图、`music.wav`、`alphatwice.txt`
- `output/`：脚本运行后生成的拼贴图、处理后音频、字母统计直方图
- `main.py`：单文件完成三项实验任务
